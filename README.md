# Semana 11 - Entrega incremental del sitio/proyecto

## Proyecto
Incremento funcional: Inscripcion a talleres comunitarios.

## Descripcion
Este proyecto permite registrar participantes en talleres comunitarios. El sistema valida nombre, edad, telefono, correo, taller y confirmacion. Si los datos son correctos, guarda la inscripcion, la muestra en una tabla, actualiza los indicadores y calcula el taller con mayor demanda.

## Como ejecutar
1. Abrir la carpeta completa en Visual Studio Code.
2. Abrir `index.html` con Live Server o directamente en el navegador.
3. Probar el formulario.
4. Abrir `tests.html` para revisar las pruebas automaticas.

## Archivos principales
- `index.html`: interfaz principal.
- `css/styles.css`: estilos visuales.
- `js/app.js`: logica JavaScript corregida.
- `tests.html`: pagina de pruebas.
- `js/tests.js`: pruebas automaticas.
- `docs/`: checklist, matriz de pruebas y bitacora.
- `evidencias/`: capturas de pantalla.
